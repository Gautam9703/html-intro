What is a Tag?

A tag is a keyword enclosed in angle brackets (< >) used to define elements in HTML. Tags help browsers understand how content should be displayed.

Two Types of Tags

  • Opening Tag
  • Closing Tag

What is an Opening Tag?

An opening tag marks the beginning of an HTML element. Some tags are self-closing and do not need a closing tag.

  • <br> Tag
  • <hr> Tag
  • <img> Tag

What is a Closing Tag?

A closing tag marks the end of an HTML element and starts with a forward slash (/).

  • <html> Tag
  • <p> Tag
  • <div> Tag

What is a Heading Tag?

Heading tags define titles and subtitles on a webpage. They range from h1 (largest) to h6 (smallest).

What is supersscripts ?.

Superscript is a small character (number, letter, or symbol) that appears slightly raised above the normal text line, used for things like exponents (\(x^{2}\)), footnotes (¹), or ordinal numbers (1st). It's the opposite of subscript, which appears below the line, and serves to provide additional meaning or reference within text, common in math, science, and computing. 
  • a2 + b 2 = a 2+ b 2+ 2ab

What is subscript?.

A subscript is a small character (number, letter, or symbol) set slightly below the normal text line, used to denote a different variable, count items in a set, specify an element in an array, or indicate atom counts in chemical formulas like \(H_{2}O\) (the '2' is a subscript). It's a form of shorthand to convey specific meaning, contrasting with superscript, which appears above the line.
